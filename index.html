ভূমিকা

Programming জগতে ভুলের কোনো স্থান নেই। সামান্য ভুলের কারণে প্রোগ্রামের ফলাফল হতে পারে সম্পূর্ণই ভুল। তবে ভুল থেকে অভিজ্ঞতা অর্জন করা যেতে পারে। এই লেখনির মূল লক্ষ্য হচ্ছে সাধারণ কিছু ভুল নিয়ে আলোচনা করা, বিশেষ করে যেসব ভুল Terminal এ কোনো error message দেয় না। Code RUN হল, অথচ কোনোরকম error message দেয়া ছাড়াই output ভুল আসছে এমন code গুলোর ভুল খুঁজে বের করা কিছুটা কঠিন। বিশেষ করে পরীক্ষার সময় যখন সল্প সময়ে code গুলো লেখতে হয়, তখন এমন ভুল খুঁজে বের করতে বেশি সমস্যা হয়।

অনেকেই পিসিতে Windows এর পাশাপাশি Ubuntu অপারেটিং সিস্টেম ইন্সটল করে। কিন্তু C++ code run করার জন্য Ubuntu লাগবেই এমন ধারণা ঠিক না। Windows এ C++ code run করানোর পাশাপাশি GNUplot ব্যবহার করা যায়, যার ব্যবহারবিধি সম্পূর্ণই Ubuntu তে terminal এ command দিয়ে GNUplot ব্যবহার করার মতই। এর ব্যবহারবিধি ও কৌশল এখানে আলোচনা করা হয়েছে। এই GNUplot যেকোনো অ্যান্ড্রয়েড স্মার্টফোনেও ব্যবহার করা সম্ভব।
